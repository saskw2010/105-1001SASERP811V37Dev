SASERP V37 - Development Roadmap

Completed Milestones (Recent)

  • **UI Modernization:**
  • Implemented a modern, accessible navigation drawer.
  • Externalized inline CSS and JavaScript from `Main.master`.
  • Upgraded to Bootstrap 5.3.2 and Font Awesome 6.
  • **Documentation:**
  • Created a comprehensive set of documentation in the `/Docs/` directory.
  • Developed an interactive, Vue-based documentation site in `/TestPages/docs/`.
  • **Code Cleanup:**
  • Refactored and cleaned up the `Main.master` page.
  • Addressed various build warnings and legacy code issues.

    • Current Work (Q3 2025)

  • **Module Enhancement:**
  • **Financials:** Improving the reporting capabilities with more flexible and customizable reports.
  • **HR:** Adding a new module for performance reviews and employee feedback.
  • **Performance Optimization:**
  • Analyzing and optimizing database queries.
  • Improving the load time of key pages and dashboards.
  • **Testing:**
  • Increasing unit test coverage for the business logic in `App_Code`.
  • Expanding the suite of automated UI tests.

    • Future Plans (Q4 2025 and Beyond)

  • **API Development:**
  • Develop a comprehensive RESTful API to allow for integration with other systems.
  • This will enable headless operation and support for modern frontend frameworks.
  • **Frontend Framework Adoption:**
  • Gradually migrate key parts of the application to a modern frontend framework like Vue.js or React.
  • This will start with the most interactive and data-heavy pages.
  • **Cloud Migration:**
  • Evaluate and plan for the migration of the application to a cloud platform like Azure.
  • This will improve scalability, reliability, and maintainability.
  • **Mobile Application:**
  • Develop a native or hybrid mobile application for key functionalities, such as dashboard access and approvals.

    • Long-Term Vision
