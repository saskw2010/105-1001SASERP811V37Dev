Self-host Font Awesome (Free) quick steps:
1) Download Font Awesome Free web package from fontawesome.com (Free).
2) Copy fa-solid-900.woff2 and fa-regular-400.woff2 into the same folder as fa-free-local.css.
3) Include <link rel="stylesheet" href="/path/to/fa-free-local.css"> in your layout.
4) Use <i class="fas fa-magnifying-glass"></i> etc in HTML, or extend CSS to add ::before icons.
